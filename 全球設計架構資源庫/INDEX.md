# 全球設計架構資源庫
## 網頁設計 → APP設計 → 整體架構規劃

> 涵蓋從網頁設計基礎到軟體架構規劃的全球最精華資源，中文和英文完整收錄。

---

## 📂 資料夾結構

| 資料夾 | 主題 | 說明 |
|--------|------|------|
| `1_網頁設計基礎` | UI設計原則、版面配置、視覺設計、色彩理論 | 網頁設計的核心基礎 |
| `2_UX使用者體驗` | 使用者研究、資訊架構、互動設計、可用性測試 | 以人為本的設計方法 |
| `3_前端開發設計` | HTML/CSS、響應式設計、CSS框架、動畫效果 | 前端技術實作 |
| `4_APP行動應用設計` | iOS/Android設計規範、行動UI、手勢設計 | 行動應用設計專區 |
| `5_設計工具與原型` | Figma、Adobe XD、Sketch、Framer、InVision | 設計工具全攻略 |
| `6_設計系統與組件庫` | Design System、組件庫、Style Guide、設計規範 | 設計系統建立指南 |
| `7_軟體架構規劃` | 系統架構、微服務、API設計、資料庫、雲端架構 | 後端架構規劃 |
| `8_產品規劃與專案管理` | 產品路線圖、敏捷開發、Scrum、需求分析 | 產品管理方法論 |

---

## 🌐 資源統計

- **總資源數量**：300+ 個全球頂級資源
- **語言涵蓋**：中文、英文、日文
- **資源類型**：平台、工具、書籍、課程、YouTube頻道、社群論壇、方法論、框架

---

## 🚀 使用建議

1. **初學者**：從 `1_網頁設計基礎` 和 `2_UX使用者體驗` 開始
2. **設計師**：重點關注 `5_設計工具與原型` 和 `6_設計系統與組件庫`
3. **開發者**：重點關注 `3_前端開發設計` 和 `7_軟體架構規劃`
4. **產品經理**：重點關注 `2_UX使用者體驗` 和 `8_產品規劃與專案管理`
5. **全端學習**：按資料夾順序依次學習
