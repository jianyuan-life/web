## 🎨 設計原則（核心知識）

| 原則 | 說明 |
|------|------|
| 近接（Proximity） | 相關元素應彼此靠近，形成視覺群組 |
| 對齊（Alignment） | 元素應在頁面上對齊，創造秩序感 |
| 對比（Contrast） | 使用差異來突出重要元素 |
| 重複（Repetition） | 重複設計元素創造一致性和品牌識別 |
| 層次結構（Hierarchy） | 透過視覺差異引導使用者注意重要資訊 |
| 格式塔理論（Gestalt） | 人類感知如何組織視覺資訊的原則 |
| 負空間（Negative Space） | 元素周圍的空白空間，定義和強調元素 |
