## 🔬 UX方法論

| 方法論 | 說明 |
|--------|------|
| 設計思考（Design Thinking） | 以人為本的解決問題方法，包含同理、定義、發想、原型、測試五個階段 |
| 使用者中心設計（UCD） | 在設計過程的每個階段都將使用者納入考量 |
| 雙鑽石模型（Double Diamond） | 探索→定義→發展→交付四個階段 |
| 精實UX（Lean UX） | 快速迭代、最小可行產品和持續學習 |
| 敏捷UX（Agile UX） | 將UX設計整合到敏捷開發流程中 |
| Google Design Sprint | 為期五天的快速解決問題和測試方法 |
| UX的五大層次 | 策略層、範疇層、結構層、骨架層、表層 |

## 🔧 UX研究方法

| 方法 | 說明 |
|------|------|
| 人物誌（Persona） | 虛構人物角色代表目標使用者群體 |
| 使用者旅程地圖（User Journey Map） | 視覺化使用者與產品互動的過程 |
| 線框圖與原型（Wireframe/Prototype） | 創建產品的低/高擬真模型 |
| 可用性測試（Usability Testing） | 讓真實使用者操作產品，收集回饋 |
